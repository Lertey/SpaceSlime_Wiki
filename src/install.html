Установка Java

Сначала нужно скачать и установить нужную для вашего ПК Java. Без неё лаунер просто не сможет запуститься. Для этого вам нужно знать что у вас за система и её разрядрость

・ Windows x64: Скачать

・ Windows x86 (x32): Скачать

・ Linux: Скачать (.deb), Скачать (.tar.gz)

・ Mac: Скачать (.dmg), Скачать (.pkg)

Теперь устанавливаем скачанный вами файл. Способ установки может отличаться в зависимости от вашей ОС. Рекомендуем перезапустить ПК.

ВНИМАНИЕ

Операционные системы Windows 7, 8, 8.1 не имеют официальной поддержки! Все проблемы которые могут возникнуть при использовании лаунчера на этих ОС не обрабатываются в support.

Установка лаунчера

После того как мы прошли первый шаг, можем установить лаунчер. Для этого скачиваем файл лаунчера и переносим его в удобное место.

После этого запускаем лаунчер и входим в свой подтверждённый аккаунт, ранее созданный на Spaceslime.fun.

Для Linux

Для запуска лаунчера рекомендуется использовать консоль.

java -jar spaceslime.jar

Если вы не установили Java а скачали архив, то java земените на путь

"{path}" -jar spaceslime.jar
